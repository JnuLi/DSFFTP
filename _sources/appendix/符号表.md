# 符号表

## 变量

|||
| :--------------: | ------------------------------------------------------------ |
| $\boldsymbol{x}$ | 表示**输入变量**，也称作**输入特征**（feature），即输入数据。 |
|      $x_i$       | 表示**输入变量**的第$i$个**特征**，即$\boldsymbol{x}={(x_1,x_2,...,x_n)}^\top$ |
| $\boldsymbol{y}$ | 表示**输出变量**，也称作**目标变量**（target），即输出数据 |

|||
|:--|:-:|
|$x$| 标量|
|$\boldsymbol{x}$|向量|
|$\boldsymbol{X}$|矩阵|
|$\mathsf{X}$|张量|
